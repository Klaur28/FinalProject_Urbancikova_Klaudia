O projekte

Informácie o aplikácii Evidencia Poistenia

O autorovi

Tento projekt vytvorila Klaudia Urbančíková v rámci finálneho projektu rekvalifikačného kurzu Tvorba www aplikací v Java SK akreditácia v roku 2025.

Popis projektu

Cieľom bolo vytvoriť komplexnú webovú aplikáciu s využitím moderných technológií a frameworkov.

Aplikácia Evidencia Poistenia slúži na správu poistencov a ich poistných zmlúv. Umožňuje jednoduché pridávanie, úpravu a odstránenie poistencov, ako aj vytváranie a správu poistných zmlúv. Aplikácia poskytuje aj základné štatistiky pre lepší prehľad o poistnom portfóliu.

Použité technológie
Java

Hlavný programovací jazyk projektu

Spring Boot

Framework pre tvorbu webových aplikácií

MariaDB

Relačná databáza pre ukladanie dát

Spring Security

Zabezpečenie aplikácie a správa používateľov

Thymeleaf

Šablónovací systém pre dynamické webové stránky

Bootstrap 5

CSS framework pre moderný dizajn

Funkcie aplikácie
Správa poistencov
  • Pridávanie nových poistencov
  • Úprava existujúcich poistencov
  • Odstránenie poistencov
  • Prehľad všetkých poistencov
Správa poistných zmlúv
  • Vytváranie nových zmlúv
  • Úprava existujúcich zmlúv
  • Odstránenie zmlúv
  • Priradenie zmluvy k poistencovi
Autorizácia a autentifikácia
  • Registrácia nových používateľov
  • Prihlasovanie používateľov
  • Odhlasovanie používateľov
Štatistiky a prehľady
  • Prehľad aktívnych zmlúv
  • Štatistiky poistencov
  • Filtrovanie a vyhľadávanie
Architektúra projektu

Projekt je postavený na architektúre Model-View-Controller (MVC) s využitím vrstveného prístupu:

Entitná vrstva

Definícia entít mapovaných na databázu (JPA)

Repozitár vrstva

Prístup k dátam (Spring Data)

Servisná vrstva

Logika služba aplikácie

Kontrolér vrstva

Spracovanie HTTP požiadaviek

Prezentačná vrstva

Zobrazenie dát používateľovi (Thymeleaf)
